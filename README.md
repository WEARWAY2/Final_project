Follow these steps to set up the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/WEARWAY2/Final_project
   ```
2. **Navigate to the project directory**
   ```bash
   cd Final_project
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```

